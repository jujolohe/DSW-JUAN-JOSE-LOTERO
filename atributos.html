ATRIBUTOS

Atributos basicos

Los elementos en HTML tienen atributos; estos son valores adicionales que configuran los elementos o ajustan su comportamiento de diversas formas para cumplir los criterios de los usuarios

Atributo Descripción
Establece un indicador único a cada elemento
Establece la clase CSS que se aplica a los estilos del elemento
Aplica de forma directa los estilos CSS de un elemento
Establece el título del elemento (Mejora la accesibilidad)

Atributos de internacionalización o I18N

Estos atributos se utilizan en aquellas páginas que muestran sus contenidos en varios idiomas y las que quieran indicar de forma explícita el idioma de sus contenidos:

Atributo Descripción
Indica el idioma del elemento
Indica el idioma del elemento, aunque tiene más prioridad que el atributo anterior y es obligatorio si se incluye el atributo lang
Indica la dirección del texto

Atributos de eventos

Estos atributos se utilizan en las páginas web que incluyen código JavaScript para realizar acciones dinámicas sobre los elementos de la página.

Pueden ser utilizados por: todos los elementos
Atributos Descripción
Ejecuta la acción cuando se realiza un clic sobre el elemento
Ejecuta la acción cuando se realiza un doble clic sobre el elemento
Ejecuta la acción cuando se detecta el botón pulsado del ratón
Ejecuta la acción cuando se detecta que se ha soltado el botón del ratón
Ejecuta la acción cuando se detecta el movimiento del ratón sobre el elemento
Ejecuta la acción cuando el ratón abandona el elemento
Ejecuta la acción cuando se detecta que el ratón se sitúa sobre el elemento

Pueden ser utilizados por: < body>
Atributo Descripción
Ejecuta la acción cuando se carga el documento
Ejecuta la acción cuando se abandona el documento
Ejecuta la acción cuando se ha modificado el tamaño de la ventana del navegador

Pueden ser utilizados por: elementos de formulario y < body>
Atributo Descripción
Ejecuta la acción cuando se detecta que la tecla esta pulsada
Ejecuta la acción cuando se detecta que se ha soltado la tecla pulsada
Ejecuta la acción cuando se pulsa una tecla

Pueden ser utilizados por: varios
Atributo Descripción Elementos que pueden usarlo
Ejecuta la acción cuando el elemento pierde el foco bien sea a través del ratón o por navegación tabulada
Ejecuta la acción cuando el elemento obtiene el foco bien sea a través del ratón o por navegación tabulada
Ejecuta la acción cuando el valor de un control ha sido modificado
Ejecuta la acción cuando el formulario es restablecido a sus valores por defecto
Ejecuta la acción cuando un usuario selecciona texto en un campo de texto
Ejecuta la acción cuando el formulario es enviado

Atributos de foco

Se le denomina foco o focus, cuando un control o elemento del documento ha sido seleccionado. Cuando ese elemento deja de estar seleccionado, "pierde el foco" y es el nuevo elemento seleccionado el que se dice que tiene "el foco".

Atributo Descripción
Establece una tecla de acceso rápido a un elemento HTML
Establece la posición del elemento en el orden de tabulación de la página (valor entre 0 y 32.767)
Indica la dirección del texto
